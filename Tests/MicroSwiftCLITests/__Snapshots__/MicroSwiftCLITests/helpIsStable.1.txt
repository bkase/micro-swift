OVERVIEW: Local-first tooling shell for MicroSwift

USAGE: micro-swift <subcommand>

OPTIONS:
  -h, --help              Show help information.

SUBCOMMANDS:
  doctor                  Print runtime/tooling diagnosis
  seed                    Seed utilities
  mlx-smoke               Run the MLX smoke check
  lex                     Run lex-only pipeline and emit structured JSON tokens
  lexergen                Generate and check lexer artifacts

  See 'micro-swift help <subcommand>' for detailed help.
